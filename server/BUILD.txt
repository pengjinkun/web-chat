Linux only (Epoll). Dependencies:

  sudo apt install build-essential cmake libssl-dev libmysqlclient-dev libhiredis-dev

Build:

  cd server
  cmake -B build -S .
  cmake --build build -j
  # import schema: mysql -u root -p < ../schema.sql
  export MYSQL_PASSWORD=...
  ./build/web_chat_server

Environment variables:

  MYSQL_HOST (default 127.0.0.1), MYSQL_PORT, MYSQL_USER, MYSQL_PASSWORD, MYSQL_DATABASE (chat_db)
  REDIS_HOST, REDIS_PORT
  PORT (WebSocket listen, default 8765)

Architecture: Boss thread blocking accept() + Worker thread epoll for clients + eventfd wake (Netty-style split).
JSON/WebSocket protocol matches the existing React client (same message types as the former Node server).
